# UI Rules

## Radius va khoang cach

- Panel/card: `8px`.
- Input/button/select/textarea: `6px`.
- Badge/status label: `999px`.
- Khoang cach giua cac field trong form: `12px - 16px`.
- Line-height mac dinh: `1.55`.

## Kieu chu

- Font mac dinh: Arial/Helvetica/sans-serif.
- Body: `14px - 16px`.
- Heading trong panel: `20px - 26px`.
- Ghi chu field: nho hon label 1-2px, mau phu, co the in nghieng.
- Ten san pham/don hang/van don: in dam.
- Gia tien: in dam, mau primary pressed.

## Mau sac

- Primary: `#2563eb`.
- Primary pressed/hover: `#1d4ed8`.
- Secondary text: `#64748b`.
- Background diu nhe: `#f6f8fb`.
- Surface: `#ffffff`.
- Border: `#d8e0ea`.
- Button: `#2563eb`.
- Button pressed: `#1d4ed8`.
- Button disabled: `#cbd5e1`.
- Error: `#dc2626`.
- Success: `#16a34a`.
- Bold text: `#111827`.

## Form

- Truong bat buoc phai co dau `*` mau do.
- Truong sai phai co vien do va thong bao loi ngay duoi field.
- Upload anh dung input file va preview, khong dung nhap URL.
- Text placeholder ngan, noi dung ro hanh dong can lam.

## Badge va status

- Badge co padding ngang, khong de chu sat vien.
- Status hien thi bang tieng Viet, khong hien ma ky thuat nhu `pending_review`, `waiting_pickup`.
- Mau status:
  - Thanh cong: xanh la nhe.
  - Can xu ly/dang chay: vang nhe.
  - Loi/huy/tu choi: do nhe.
  - Luu tru/khong hoat dong: xam nhe.

## Tien Viet

- Dinh dang mac dinh: `500.000 đ`, `1.000 đ`, `1.000.000 đ`, `1.212.000.000 đ`.
- Dung dau `.` sau moi 3 chu so.
- Gia tien trong bang/list dung class `price-cell`.

## Menu

- Nen menu sang diu, co border nhe.
- Link menu co the xuong dong khi man hinh hep.
- Khi menu nhieu cap, cap cha nen in dam, cap con can thut/nhom ro rang.
